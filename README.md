# **Проект Mesto**
## Место для обмена фотографиями

Проект сделан при исползовании технологий:
* Флекс-бокс верстка;
* Грид-сетки;
* Адаптивная верстка под смартфоны, планшеты и Desktop;
* Создание popup.
